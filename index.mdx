  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get ROS2 up and running in three simple steps
  </p>

  <Steps>
    <Step title="Install the CLI tool">
      Install via Homebrew for the easiest setup:
      
      ```bash
      brew tap neelashkannan/ros-installer
      brew install ros2-installer
      ```
      
      Or manually with Python:
      
      ```bash
      git clone https://github.com/neelashkannan/ROS2-Installer-CLI.git
      cd ROS2-Installer-CLI
      pip3 install -r requirements.txt
      ```
    </Step>

    <Step title="Run the installer">
      Launch the interactive installer to choose your ROS2 distribution and package set:
      
      ```bash
      # Interactive mode (recommended for first-time users)
      ros2-installer
      
      # Or use silent mode for automation
      ros2-installer --silent --ros-distro humble --package-set desktop
      ```
      
      <Info>
        On Ubuntu, the installer will install ROS2 natively via apt. On macOS and other systems, it uses Docker automatically.
      </Info>
    </Step>

    <Step title="Verify installation">
      Once installation completes, verify ROS2 is working:
      
      ```bash
      # Ubuntu
      source ~/.bashrc
      ros2 --help
      
      # macOS / Docker
      ros2 topic list
      ```
      
      You should see ROS2 commands available and ready to use.
    </Step>
  </Steps>
</div>
